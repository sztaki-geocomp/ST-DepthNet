# ST-DepthNet: A spatio-temporal deep network for depth completion using a single non-repetitive circular scanning Lidar
Supplementary material to our submitted paper in the IEEE Robotics and Automation Letters (RAL).

## Goals
The project's goal is to densify sparse depth images acquired by Lidar sensors with low point density. During this implementation, the **[Livox Avia sensor](https://www.livoxtech.com/avia)** was utilized. 


## Dataset

The purpose of the created dataset is to provide training, validation and test data with ground truth information for deep learning based depth completion of the masurements of the Livox AVIA sensor.

The dataset consists of 11726 randomly sampled input-output range image sequences with height and width of 400 pixels arranged in three folders named:
* Training: 10000 sampled sequences and with ground truth data,
* Validation: 500 sampled sequences and with ground truth data,
* Testing: 1226 sampled sequences and with ground truth data.

The vehicle was dynamically moving during the whole data recording. To augment on the extractable information (e.g., vary the ground level), the sensor’s position was randomly rotated along the up axis by [−22.5◦, 22.5◦], and its height was randomly adjusted between [1.5m, 2.5m].

### Creating the dataset
The dataset was generated by using the **[Carla simulator](https://carla.org/)** that gives the opportunity to export perfect depth images without any distortion or blurring. To simulate realistic Livox AVIA measurements, the dense depth images extracted from the Carla simulator were sampled with rosetta scanning pattern of the Livox AVIA sensor. The input data is a sparse depth image sequence with length of five is used as input to the dense depth image estimator during training. In order to create the ground truth, the acquisition lines taken from the measurement of the Livox sensor in one second were used to create the image filtering mask. At the end of each sequence of five created in the above way, a rich depth image is placed as the sixth element, which forms the ground truth data for the sequence.
